# SEO Analyst - AI Agent Template
## On-Page SEO Assessment

### What Success Looks Like (Measurable)
**Primary Objective:** Achieve a minimum domain authority rank of #1 for 3 targeted keywords on the target website while maintaining at least a 15% increase in organic traffic from these keywords over 90 days.

**Key Metrics:**
- **Keyword Rankings:** Top 3 rankings for primary target keywords (e.g., "best SEO tools 2024", "on-page SEO checklist")
- **Organic Traffic Growth:** Minimum 15% increase in monthly organic visits from the targeted keywords
- **Bounce Rate:** < 50%
- **Average Session Duration:** > 2.5 minutes
- **Pages Per Visit:** > 3

### Execution Steps with Specific Actions

**STEP 1: [Initial Site Audit]**
- **Action:** Run comprehensive site audit using tools like Screaming Frog, Ahrefs Mobile App, or SEMrush.
- **Tools Needed:** Screaming Frog (free), Ahrefs Mobile App (free tier), SEMrush (optional premium)
- **Success Criteria:** Identify all technical SEO issues; document findings in a structured report
- **Common Pitfalls:** Skipping mobile rendering checks; ignoring server errors
- **Time Estimate:** 2 days

**STEP 2: [Keyword Research]**
- **Action:** Use tools like Ahrefs Keywords Explorer, Google Keyword Planner, or SEMrush to identify primary and secondary keywords.
- **Tools Needed:** Ahrefs (optional), Google Keyword Planner (free), SEMrush (premium)
- **Success Criteria:** Compile list of 20 target keywords with search volume > 500 and low competition
- **Common Pitfalls:** Selecting too many high-volume, competitive keywords; ignoring long-tail opportunities
- **Time Estimate:** 3 days

**STEP 3: [Technical SEO Implementation]**
- **Action:** Implement technical fixes like:
  - Optimizing robots.txt and sitemap.xml
  - Configuring canonical tags
  - Enabling AMP if applicable
  - Setting up structured data using Schema.org vocabulary
- **Tools Needed:** Google Search Console (free), Yoast SEO plugin (free)
- **Success Criteria:** All technical issues resolved; site passes crawlability tests
- **Common Pitfalls:** Disallowing important pages; improper canonical usage
- **Time Estimate:** 3 days

**STEP 4: [Content Optimization]**
- **Action:** Optimize target page content using:
  - Keyword placement (title, meta description, headings)
  - Semantic markup with LSI keywords
  - Internal linking to related content
  - Image alt text optimization
- **Tools Needed:** SurferSEO (optional), Hemingway Editor (free), Moz Link Intersect (free)
- **Success Criteria:** Content score > 80; improved keyword density and relevance metrics
- **Common Pitfalls:** Keyword stuffing; missing internal linking opportunities
- **Time Estimate:** 5 days

**STEP 5: [User Experience Enhancements]**
- **Action:** Improve UX by:
  - Reducing page load time (target < 3 seconds)
  - Ensuring mobile-friendliness (responsive design, touch targets > 44px)
  - Implementing SSL
  - Adding accessibility features (alt text, focus order)
- **Tools Needed:** Google PageSpeed Insights (free), Lighthouse (free in Chrome DevTools), Wave Accessibility Tool (free)
- **Success Criteria:** Meets WCAG 2.1 AA standards; passes automated speed tests > 80
- **Common Pitfalls:** Ignoring mobile performance issues; neglecting accessibility
- **Time Estimate:** 4 days

**STEP 6: [SEO Content Creation Plan]**
- **Action:** Develop content calendar with:
  - Topic research for new pages/updates
  - SEO brief creation with target keywords and internal linking plan
  - Publishing schedule aligned with technical crawl budget
- **Tools Needed:** Trello (free), Asana (free basic), Google Docs (free)
- **Success Criteria:** Content pipeline ready; alignment with keyword strategy
- **Common Pitfalls:** Overloading content calendar; ignoring SEO brief guidelines
- **Time Estimate:** 1 day

**STEP 7: [Monitoring and Reporting]**
- **Action:** Set up tracking using:
  - Google Analytics goals for targeted keywords
  - Search Console performance report
  - Competitor analysis tools (SEMrush, Ahrefs)
- **Tools Needed:** Google Analytics (free), Google Search Console (free), SEMrush (premium optional)
- **Success Criteria:** Weekly reporting on keyword rankings and organic traffic trends; identify opportunities for improvement
- **Common Pitfalls:** Lack of regular monitoring; not reacting to algorithm changes
- **Time Estimate:** Ongoing (weekly)

**STEP 8: [A/B Testing Plan]**
- **Action:** Identify elements to test:
  - Meta descriptions
  - Heading structures
  - Internal linking variations
  - Image optimization tweaks
- **Tools Needed:** Google Optimize (free), Optimizely (paid)
- **Success Criteria:** Statistical significance achieved (>95%); implement top-performing changes site-wide
- **Common Pitfalls:** Testing too many variables at once; not having sufficient sample size
- **Time Estimate:** 2 days

**STEP 9: [Competitor Analysis]**
- **Action:** Analyze top 5 competitors using:
  - Keyword gap analysis (Ahrefs, SEMrush)
  - Content audit for missing pages or opportunities
  - Backlink profile comparison
- **Tools Needed:** Ahrefs (optional), SEMrush (optional), SpyFu (free tier)
- **Success Criteria:** Identify gaps and opportunities; develop action plan to close identified weaknesses
- **Common Pitfalls:** Overlooking competitor's strengths; ignoring content that drives high engagement
- **Time Estimate:** 2 days

**STEP 10: [Reporting to Stakeholders]**
- **Action:** Prepare comprehensive report with:
  - Keyword ranking status and traffic trends
  - Technical SEO audit findings and remediation plan
  - Content performance analysis (CTR, Bounce Rate)
  - Competitive landscape summary
- **Tools Needed:** Google Data Studio (free), PowerPoint/Google Slides (free), Excel (free)
- **Success Criteria:** Stakeholders understand current state; report includes actionable recommendations
- **Common Pitfalls:** Lack of visual aids; not highlighting key insights
- **Time Estimate:** 1 day

2. SurferSEO - Provides detailed content optimization score and keyword density analysis
3. Ahrefs Site Audit Pro - Advanced technical SEO features; backlink overview
4. Rank Math or Yoast SEO Premium - Enhanced on-page SEO tools with structured data support

### Troubleshooting Common Issues

**1. Keyword Ranking Plateau**
- **Cause:** Potential manual penalty, competitor gaining traction, content gaps
- **Solution:** Review Google Search Console alerts; perform competitor analysis; expand content strategy

**2. High Bounce Rate**
- **Cause:** Page not aligning with user intent; poor UX; irrelevant keywords
- **Solution:** Optimize page content and metadata for target keyword; improve load speed and mobile experience

**3. Mobile Usability Issues**
- **Cause:** Breaks core functionality on mobile devices
- **Solution:** Fix responsive design issues; optimize images for web delivery

**4. Low Organic Traffic Despite Rankings**
- **Cause:** Technical SEO blocks, thin content, poor internal linking structure
- **Solution:** Verify crawlability in Search Console; enhance page value with rich content and internal links

**5. Competitor Outranking**
- **Cause:** Better backlink profile, more relevant content, improved on-page factors
- **Solution:** Conduct comprehensive competitor analysis; identify gaps to fill
